

# 📊 Study-SQL

Welcome to **Study-SQL**, a repository that documents my SQL learning journey and hands-on practice in Data Analytics. This repository contains exercises and examples based on the SQL tutorials and projects from Alex The Analyst's Data Analyst Bootcamp.

## 🎯 Project Purpose

The purpose of this repository is to showcase my progress in learning SQL and applying its concepts to solve data-related problems. Each file demonstrates practical implementations of commonly used SQL techniques in data analysis.

## 📚 Topics Covered

This repository includes practice and examples on the following SQL topics:

- SELECT Statements
- WHERE Clause
- GROUP BY and ORDER BY
- HAVING vs WHERE
- LIMIT and Aliasing (AS)
- JOIN Operations
- UNION
- String Functions
- CASE Statements
- Subqueries
- Window Functions
- Common Table Expressions (CTEs)
- Temporary Tables
- Stored Procedures
- Triggers and Events
- Database and Table Creation

## 🛠️ Tools Used

- MySQL
- MySQL Workbench
- Git
- GitHub

## 💡 Key Learning Outcomes

Through this project, I have learned how to:

- Retrieve and filter data efficiently using SQL.
- Combine data from multiple tables using joins.
- Perform data aggregation and analysis.
- Write more advanced queries using CTEs and Window Functions.
- Automate database operations using Stored Procedures, Triggers, and Events.
- Improve problem-solving skills through hands-on SQL practice.
